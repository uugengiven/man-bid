Upon completion of this course/week, the student should be able to demonstrate at least average ability in the following areas:

## Sendmail/Linux Email
* Install and Configure Email Server
* Configure POP3 and SMTP in Linux
* Configure IMAP mailboxes in Linux
* Create user mailboxes
* Setup Anti-Spam measures

## MySQL
* Install and Configure MySQL
* Setup security for MySQL
* Setup network settings in MySQL
* Create databases and users in MySQL
* Create backup plans for MySQL

## Troubleshooting and Monitoring
* Explain common logging scenarios
* Setup and use Nagios for monitoring
* Solve common errors in nginx/mysql/postgres/linux kernel
